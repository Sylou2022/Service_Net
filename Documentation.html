Documentation du Site



Ce document est une explication complète du Site. Nous allons trouver la procédure de recherche des services.
Dès la page d'accueil, il est possible d'apercevoir les services proposés sous images défilantes.


Les différentes étapes à suivre:


Cliquez sur le bouton Service sur la barre de Menu de l'Accueil

Ensuite, affichez la page des services, puis choisissez votre services dans le combobox

Un message de confirmation vous sera présenté

Cliquez sur ok pour afficher la page des professionnels
Cette page de professionnels sert à vérifier la proximité des agents.


Si le choix n'est pas effectué vous ne pourrez pas affichez la page des professionnels

Avec la barre de recherche, saisissez votre ville.

Après recherche de proximié, cliquez sur l'icone Agenda

Pour afficher l'Agenda de Disponibité des professionnels


Si le professionnel du métier concerné selon votre demande

est disponible, cliquez sur le bouton WhatsApp

Et contactez directement l'administrateur pour prendre RDV


Votre débit peut se faire en contact avec le professionnel après RVD validé

Votre facture peut etre envoyé par email ou récupérer après service

Il est également possible de garder vos factures dans l'historique


Créez votre votre compte pour garder vos documents

Contactez le responsable en cas de mot de passe oublié

Envoyez un commentaire au responsable après service
